# Описание

Веб-приложение для демонстрации работы с Bootstrap 5. Проект включает простой HTTP-сервер на Python и HTML-страницы с адаптивным дизайном, реализованным с помощью фреймворка Bootstrap.

## Структура проекта
```
bootstrap_demo/
├── templates/          # HTML-файлы
│   ├── contacts.html  # Страница контактов
│   └── catalog.html   # Страница каталога
├── main.py            # HTTP-сервер
└── README.md          # Документация
```

## Функционал
- Отображение HTML-страниц с Bootstrap-компонентами
- Прием и обработка POST-запросов от форм
- Вывод данных формы в консоль сервера
- Адаптивный дизайн с боковым меню и навигацией

## Технологии
- Python 3 (http.server)
- Bootstrap 5.0.2
- HTML5, CSS3

## Запуск проекта
```bash
python main.py
```
После запуска сервер будет доступен по адресу: http://localhost:8080

# Краткое описание страниц

## contacts.html
Страница контактов с формой обратной связи. Содержит:
- Темную навигационную панель с логотипом
- Боковое меню с разделами (Главная, Категории, Заказы, Контакты)
- Выпадающий список пользователя внизу меню
- Форму обратной связи с полями:
  - Имя (обязательное поле)
  - Почта (обязательное поле, валидация email)
  - Сообщение (текстовая область, обязательное поле)
- Блок с информацией о контактах
- Отправка данных формы через POST-запрос на сервер

## catalog.html
Страница каталога товаров. Содержит:
- Темную навигационную панель с логотипом
- Боковое меню с разделами
- Выпадающий список пользователя
- 4 карточки товаров, расположенные в 2 ряда (2×2)
- Каждая карточка включает:
  - Серый блок-заглушку "Thumbnail"
  - Название товара
  - Описание товара
  - Кнопки "View" и "Edit"
  - Время чтения (в минутах)
